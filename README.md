# Shabong — Cock Fight Betting Game

A 2D cock fight betting game built with Vue 3 and Phaser 3, **single-player in the browser** (no server or network required for gameplay).

## Features

- **Betting**: Place bets on MERON (red cock) or WALA (white cock)
- **Timed fights**: HP-based combat with countdown (see `shared/fightConstants.js` for duration)
- **Random attacks**: Dock, scratch, fly + scratch / critical
- **Rustic UI**: Wooden frame, balance display, crowd atmosphere audio

## Setup

```bash
npm install
npm run dev
```

Open the URL Vite prints (usually `http://localhost:5173`).

## Tech Stack

- Vue 3 + Pinia
- Phaser 3
- Vite
